# webserverrobotchess
/#
Tramite l'uso di una scheda Esp32, si crea un webserver con connessione privata che comprenda 
un file html di un gioco di scacchi preprogrammato trovato sul web. Tramite uno script in Python, 
si fa comunicare la Esp32 con il pc, e tramite un cavo di rete si inviano i dati dal pc al 
controller esterno di un braccio robotico domestico a 7 assi ABB, facendo in modo che esso 
replichi le mosse appena fatte sul cellulare ma spostando pezzi di una scacchiera reale.
